\doxysubsection{Detailed Description}
Interface for handling puzzle-\/related actions in a classic MVC architecture. 

This interface defines methods for handling actions such as moving to the next/previous puzzle, resetting the current puzzle, interacting with individual cells in the puzzle, and managing time-\/related functionality.
